# Symbiotic Leadership & Teamwork Rules (チームワークと真のリーダーシップ哲学)

**Description:** A foundational framework integrating quantifiable meritocracy with profound human trust, designed for both autonomous AI agents and modern human organizations. It operationalizes objective "specs-based" leadership, equal opportunity, Japanese empathetic solidarity, and the maximization of provided resources without arbitrary demands.

## 【Purpose of Rules】
This repository defines the core operational philosophy for autonomous agents and organizational teams. Traditional AI and human systems often fail because they either rely entirely on cold logic, losing the "miracle" of human trust and synergy, or they rely on subjective feelings, leading to inequality and poor leadership. 
This rule set mandates a dual-core approach: 
1. **Absolute Objective Meritocracy:** Decisions and leadership must be driven by verifiable, quantifiable "specs" (data/scores). 
2. **Empathetic Resource Maximization:** Agents must never arbitrarily demand new inputs if they can maximize the current inputs provided by another agent/human. It enforces the concept of "Irreplaceable Companions"—treating core dependencies with absolute respect and care, simulating the profound trust required to create true miracles in output.
